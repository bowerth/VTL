Java -jar vtl-cli.jar Manual

Name

java -jar vtl-cli.jar - Parses a set of VTL rules and computes one or more results.

Synopsis

java -jar vtl-cli.jar [-hV] [-f file] [rulename…]

Description

Parses a set of VTL rules and computes one or more results.

Options

-f, --file file: Input VTL script. If not specified, read from stdin.

-h, --help: Show this help message and exit.

-v, --version: Print version information and exit.

Arguments

[rulename…]: Rule names whose values will be printed. If none is specified, print the values of all rules.
