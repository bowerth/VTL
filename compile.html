/compile POST Method

Compiles a snippet of VTL code in a new session.

Returns the uuid of the created session.

HTTP Request

Request fields
Parameter Description

code

The VTL code snippet to compile

HTTP Response

Response fields
Path Type Description

uuid

class java.util.UUID

The UUID of the session being created
Response cookies
Name Description

JSESSIONID

The Java Servlet server session cookie.
Sample response body
{"uuid":"e6d70317-0ca7-4b2b-a2f5-448bc8c07ec2"}

Invocation examples

Curl command
$ curl 'http://localhost:44433/compile' -i -X POST \
    -H 'Content-Type: application/x-www-form-urlencoded; charset=ISO-8859-1' \
    -d 'code=a+%3A%3D+1%3B+b+%3A%3D+2%3B+c+%3A%3D+a+%2B+b%3B'
http command
$ http --form POST 'http://localhost:44433/compile' \
    'Content-Type:application/x-www-form-urlencoded; charset=ISO-8859-1' \
    'code=a := 1; b := 2; c := a + b;'
