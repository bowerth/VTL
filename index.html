VTL E&E

The prototypal engine and editor for the Validation and Transformation Language

Get Started

VTL E&E

is an interpreter engine and an editor for the Validation and Transformation Language, a declarative, metadata-driven language with the goal of ensuring uniform data quality across entities.

Developed by Bank of Italy within the SDMX Tools Task Force of the Statistics Committee (STC), an organism of the Working Group on Statistical Information Management (WGSIM).

Minimalist design

Just install the R package to feature a full-fledged VTL code editor with VTL syntax highlighting and code suggestions.

Browse the results as you write, via just-in-time rule evaluation, and keep your model clean with a top-down view of dependencies between the rules.

Starting with the editor is as easy as installing the R package: no need for any other tool, such as databases, web servers, compilers or others.
