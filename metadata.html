/metadata GET Method

Returns the metadata associated with a given object defined in a given session.

HTTP Request

Request fields
Parameter Description

uuid

The UUID of the VTL session

alias

The alias of the object whose value is returned
Request cookies
Name Description

JSESSIONID

The Java Servlet server session cookie.

HTTP Response

Response fields
Path Type Description

[].domain

String

The VTL domain of a scalar value or of a component of a dataset

[].name

String

If the object is a dataset, the name of the component

[].role

String

If the object is a dataset, the role (IDENTIFIER, MEASURE, ATTRIBUTE) of the component
Sample response body
[{"domain":"integer"}]

Invocation examples

Curl command
$ curl 'http://localhost:44433/metadata?alias=c&uuid=e6d70317-0ca7-4b2b-a2f5-448bc8c07ec2' -i -X GET \
    -H 'Accept: application/json' \
    --cookie 'JSESSIONID=9F9D09BC3505A208DF90ED0D34AF9F5C'
http command
$ http GET 'http://localhost:44433/metadata?alias=c&uuid=e6d70317-0ca7-4b2b-a2f5-448bc8c07ec2' \
    'Accept:application/json' \
    'Cookie:JSESSIONID=9F9D09BC3505A208DF90ED0D34AF9F5C'
